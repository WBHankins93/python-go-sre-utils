## Setup

Each project has its own README and can be run independently.

#### For Python-based tools:

```bash
cd log-alert-agent
pip install -r requirements.txt
python agent.py ...
```

#### Go-based tools can be run via:

```bash
cd some-go-tool
go run main.go
```
